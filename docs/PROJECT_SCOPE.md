# SAR-CCTV 다중 센서 융합 기반 전차 위협 인지 시스템 — 프로젝트 범위

---

## HOME: 시스템 진입

- 로그인 기능: 사용자 인증, 로그인 기록 저장
- 회원가입 기능: 기본 정보 입력, 필수 인증 약관 동의
- 보안 강화: 비밀번호 찾기, 기기 신뢰 여부 선택 가능
- 로그인 여부에 따른 시스템 접근 권한 조정

---

## Page 1: 광역 감시 및 변화 탐지

- SAR 위성·SCL/CCD 기반 전천후 탐지: 기상 상태와 관계없이 전차 탐지
- 지형 분석: SCL 데이터로 기동 불가 지역 식별, 예상 이동 경로 1차 압축
- CCD 기반 기동 판단: 이전 데이터와 정밀 비교, Coherent 변화로 전차 기동 여부 확정
- 변화 탐지 결과와 지도 연동 → 변화 영역 500m 이내 접근 시 경고
- 방해 요소 아이콘/마커를 지도에 표시, 클릭 시 세부 위험 정보 확인 가능

---

## Page 2: 정밀 식별 및 자세 분석

- SAR 좌표와 인근 CCTV 연동으로 전차 고해상도 포착
- AI 기반 전차 기종 식별: YOLO 검출·분류 (90식, K1A1, K2, M1A2, T-90a, tiger 등)
- 3D 매칭: 식별된 전차 종의 3D 모델과 실시간 영상 대조
- 포신 각도 분석: Yaw 방향(본체 향각), 앙각(포신 상하 각도) 정밀 계산
- 방해 요소 마커 클릭 시 세부 정보 및 삭제 여부 선택 가능 → 동적 경로 재탐색 반영

---

## Page 3: 위험 지도 및 결심 지원

- 사용자 입력 조건(거리, 시간, 위험 회피율)에 따라 최적 경로 제공
- 전차 제원·포신 각도·지형 반영 탄도 추정 → 예상 탄착 지점 산출
- 위험 구역을 지도상에 색상으로 구분 표시 (D* Lite, CCH+A*, RTAA* 등)
- 탐색 결과는 이동 거리, 예상 소요 시간으로 정량적 비교 지원
- 아군 부대 회피 기동 유도, 생존성 극대화를 위한 실시간 결심 지원

---

## Page 4: 사용 가이드

- 시스템 주요 기능(광역 감시, 정밀 식별, 위험 지도, 경로 탐색 등)을 직관적 UI로 소개
- 사용자가 서비스를 쉽게 이해할 수 있도록 단계별 안내 화면 제공
- 직관적인 경고·안내 아이콘을 통해 상황 인식 및 대응력 향상
